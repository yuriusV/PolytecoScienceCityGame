# PolytecoScienceCityGame
Game for game development course

> Installation
```bash

# compile

git clone https://github.com/yuriusV/PolytecoScienceCityGame

cd PolytecoScienceCityGame

npm install

cd rn

npm install

react-native run-android

# Run via expo

# at root folder 
cd expo

npm install -g expo-cli

npm install

npm start


```
